Welcome to pyTensorRT's documentation!
======================================

**pyTensorRT** is a Python library for working with TensorRT. `NVIDIA® TensorRT™ <https://developer.nvidia.com/tensorrt#:~:text=TensorRT%2C%20built%20on%20the%20NVIDIA,and%20others%20on%20NVIDIA%20GPUs.>`_.  is an SDK for high-performance deep learning inference, includes a deep learning inference optimizer and runtime that delivers low latency and high throughput for inference applications.  
to install this module see chapter :doc:`installation`
